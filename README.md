## Clases de equivalencia modulo 'ingreso'

| Campo    | Clases validas                          |clases invalidas |
| ---------- | --------------------------------------|-----|
| Usuario | 5 <= usuario <= 15 | usuario < 5; usuario > 15; ususario = null| 
| Contraseña | contraseña >=| contraseña < 8 ; contraseña = null|
